# TeachFlow

Plataforma de gestão para professores — backend FastAPI + Supabase, organizada como monorepo preparado para apps (frontends) e packages (libs compartilhadas).

## Estrutura

```
TeachFlow/
├── apps/         # frontends (web, mobile)
├── backend/      # API FastAPI
└── packages/     # libs compartilhadas
```
